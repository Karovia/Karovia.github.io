项目概述

U_learner 是一个基于纯前端技术栈的在线学习平台，采用模块化设计，支持多种AI服务集成。用户可以输入学习内容，系统会调用AI生成PDF文档、思维导图和动画演示等多种格式的学习材料。

核心特点

  • 多AI服务支持：集成OpenAI、Google Gemini、Claude等7种AI服务
  • 多种学习方式：支持PDF文档、思维导图、动画演示三种输出格式
  • 响应式设计：完美适配桌面和移动设备
  • 本地存储：API密钥和用户设置保存在浏览器本地
  • 一键生成：自动整理学习资料并生成多种格式

数据流设计

搜索流程：

  1. 用户输入学习内容
  2. 表单验证和API密钥检查
  3. 调用对应的AI服务API
  4. 生成学习内容
  5. 保存结果到sessionStorage
  6. 跳转到结果页面展示

UI/UX设计

设计原则

  • 一致性：统一的色彩、字体和间距系统
  • 响应式：自适应不同屏幕尺寸
  • 可访问性：支持键盘导航和屏幕阅读器
  • 反馈机制：即时的视觉和文字反馈

交互设计

安全设计

数据安全

  • API密钥保护：仅存储在localStorage，不通过网络传输
  • 输入验证：前端格式验证和后端API密钥验证
  • XSS防护：对用户输入进行过滤和转义
  • CSP策略：限制资源加载来源，禁用不必要的内容

GitHub Pages部署步骤

  1. 创建GitHub仓库（必须是Public仓库）
  2. 将代码推送到仓库
  3. 进入仓库Settings → Pages
  4. 选择Source为"Deploy from a branch"
  5. Branch选择"main"，目录选择"/(root)"
  6. 点击Save，几分钟后即可访问
